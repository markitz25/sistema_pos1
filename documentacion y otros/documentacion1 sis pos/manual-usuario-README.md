# Manual de Usuario del Sistema POS

Autores: **Marco Saavedra** | **Velásquez Emanuel**

## Objetivo
Guía práctica para usar el Sistema POS: entender pantallas, roles y flujos habituales (ventas, inventario, clientes y reportes) sin requerir conocimientos técnicos.

## Público objetivo
- Administradores del negocio/tienda.
- Cajeros o personal que registra ventas.
- Encargados de inventario y reportes.

## Qué cubre
- Qué es el Sistema POS y sus módulos.
- Instalación básica en un entorno de pruebas.
- Roles, accesos y permisos clave.
- Flujos principales: inventario, clientes, ventas, reportes y facturas.
- Exportaciones y tablero (dashboard).
- Preguntas frecuentes y buenas prácticas básicas.

Para detalles de infraestructura o código, considerar un manual técnico separado.
