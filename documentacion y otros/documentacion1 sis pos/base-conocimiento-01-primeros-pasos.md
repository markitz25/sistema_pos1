# Primeros pasos con el Sistema POS

## 1. Primer acceso
1. Pedir al administrador un usuario y contraseña.
2. Abrir el navegador y acceder a la URL del sistema.
3. En /login/, ingresar credenciales.
4. Una vez dentro, cambiar la contraseña desde el menú de perfil.

## 2. Configuración inicial recomendada
- Crear categorías principales de productos (ej: Lácteos, Bebidas, Aseo).
- Registrar primero los productos más vendidos.
- Registrar clientes frecuentes si se desea historial de compras.
- Crear usuarios para cada trabajador con el rol adecuado.

## 3. Flujo básico diario
1. El trabajador inicia sesión.
2. Revisa stock de productos clave si es necesario.
3. Registra ventas durante el día en /ventas/.
4. Al cierre del turno:
   - Cierra sesión.
   - El administrador revisa el reporte del día en /reportes/ventas/ o el tablero.
