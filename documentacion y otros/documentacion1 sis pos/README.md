# Sistema POS — Documentación General

Autores: **Marco Saavedra** | **Velásquez Emanuel**

Esta carpeta reúne la documentación funcional y operativa del Sistema POS. Incluye el manual de usuario, la descripción de las APIs internas y una base de conocimientos con buenas prácticas y resolución de problemas.

## Alcance de esta documentación

- Explicar pantallas y flujos principales para administradores y trabajadores.
- Describir las rutas API que devuelven JSON usadas por el frontend (AJAX) y posibles integraciones internas.
- Reunir buenas prácticas, primeros pasos y soluciones rápidas a problemas frecuentes.
