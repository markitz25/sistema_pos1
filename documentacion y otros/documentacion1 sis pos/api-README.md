# Documentación de la API del Sistema POS

Autores: **Marco Saavedra** | **Velásquez Emanuel**

APIs internas que devuelven JSON para el frontend (AJAX) y para integraciones internas. Todas las rutas requieren sesión iniciada (`login_required`) y algunas validan rol de administrador.

## Archivos por módulo
- `api-ventas.md`
- `api-clientes.md`
- `api-inventario.md`
- `api-tablero.md`

En cada archivo se indican rutas, parámetros, ejemplos de respuesta y notas de permisos/errores.
