# 02. DIAGRAMAS DEL SISTEMA

Esta carpeta contiene los diagramas de casos de uso del Sistema POS en formato Draw.io.

## Archivos Incluidos

### 1. diagrama_casos_uso_completo.drawio
Diagrama completo de casos de uso que muestra:
- Todos los actores del sistema (Administrador, Trabajador, Cliente)
- Todos los casos de uso de cada módulo:
  - Autenticación (5 casos de uso)
  - Gestión de Trabajadores (5 casos de uso)
  - Gestión de Clientes (7 casos de uso)
  - Gestión de Inventario (7 casos de uso)
  - Ventas (12 casos de uso)
  - Reportes (10 casos de uso)
- Relaciones entre actores y casos de uso
- Límites del sistema

**Total de casos de uso:** 46

### 2. diagrama_casos_uso_optimizado.drawio
Diagrama optimizado y simplificado que muestra:
- Vista de módulos principales agrupados
- Casos de uso esenciales del sistema
- Relaciones include y extend
- Enfoque en funcionalidades principales (Punto de Venta, Dashboard)
- Mejor visualización para presentaciones ejecutivas

## Cómo Usar los Diagramas

1. **Para editar:** Abrir los archivos .drawio en [diagrams.net](https://app.diagrams.net/) o Draw.io Desktop
2. **Para visualizar:** Importar en Draw.io o usar visualizador compatible
3. **Para exportar:** Desde Draw.io se puede exportar a PNG, JPG, PDF, SVG

## Formato
- Formato: Draw.io XML
- Compatible con: Draw.io, diagrams.net
- Editable: Sí
